✨ 功能清单

🔐
服务器登录
输入 Navidrome 服务器地址、用户名、密码进行登录。支持记住地址和自动登录。
🏠
首页推荐
最新专辑、每日推荐、最近播放、随机专辑、歌单、电台六大板块。
🎵
全功能播放器
大封面显示，左滑歌词，右滑推荐歌曲。支持随机/顺序/单曲循环三种播放模式。
🔍
全局搜索
搜索歌曲、歌手、专辑、歌单。支持热门搜索和搜索历史。
❤️
收藏管理
收藏歌曲、专辑、歌手、歌单。分类查看，快速访问。
⏰
定时停止
支持 15分钟、30分钟、1小时、2小时定时停止播放，适合睡前听歌。
💾
缓存管理
可设置 2/4/8/16GB 缓存大小，查看缓存占用，一键清除缓存。
📋
歌单管理
查看和管理 Navidrome 服务器上的歌单，添加歌曲到歌单。

🛠️ 技术方案

开发语言：Kotlin
UI框架：Jetpack Compose (Material 3)
架构：MVVM + Repository Pattern
网络：Retrofit + OkHttp (Subsonic API)
图片加载：Coil
音频播放：ExoPlayer (Media3)
本地存储：Room Database + DataStore
缓存：自定义 LRU 缓存 + 文件缓存
最低版本：Android 8.0 (API 26)
目标版本：Android 14 (API 34)
