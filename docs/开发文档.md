# LeChenMusic Player 开发文档

## 1. 项目概述

LeChenMusic Player 是一个基于 Kotlin + Jetpack Compose 的 Android 音乐+有声书播放器，配套 [LeChenMusic-Server](https://github.com/yueyoue/LeChenMusic-Server) 使用。

- **语言：** Kotlin
- **UI 框架：** Jetpack Compose + Material 3
- **架构：** MVVM（ViewModel + StateFlow）
- **网络：** Retrofit + OkHttp
- **图片加载：** Coil
- **播放器：** Media3 ExoPlayer
- **最低 SDK：** Android 8.0 (API 26)

### 4.2 有声书封面 URL

```kotlin
fun getAudiobookCoverUrl(serverUrl: String, username: String, password: String, bookId: String): String {
    val normalizedUrl = serverUrl.trimEnd('/')
    val encodedPass = if (password.startsWith("enc:")) password
                      else "enc:${password.toByteArray().joinToString("") { "%02x".format(it) }}"
    return "$normalizedUrl/api/audiobook/$bookId/cover?u=$username&p=$encodedPass"
}
```

## 8. 播放器架构

### 8.1 音乐播放

```kotlin
// MusicPlayerManager.kt
fun playSong(song: Song, songs: List<Song>) {
    // 创建 MediaItem 列表，使用 Subsonic API 流式播放
    val mediaItems = songs.map { s ->
        val url = repository!!.getStreamUrl(s.id)
        MediaItem.Builder().setUri(url).setMediaId(s.id).build()
    }
    player?.setMediaItems(mediaItems, index, 0)
    player?.prepare()
    player?.play()
}
```

### 8.2 有声书播放

```kotlin
fun playUrl(url: String, title: String, artist: String, mediaId: String) {
    val mediaItem = MediaItem.Builder()
        .setUri(url)
        .setMediaId(mediaId)
        .setMediaMetadata(
            Media3Metadata.Builder()
                .setTitle(title)
                .setArtist(artist)
                .setAlbumTitle("有声书")
                .build()
        )
        .build()
    player?.setMediaItem(mediaItem)
    player?.prepare()
    player?.play()
}
```

## 10. 服务端配合

### 10.1 有声书媒体库配置

1. Docker 挂载有声书目录：`-v /你的有声书目录:/audiobooks:ro`
2. Web 后台 → 媒体库 → 新增媒体库
3. 路径填写 `/audiobooks`，类型选择「有声书」

### 10.3 有声书扫描器 Bug（已修复）

**问题：** 全量扫描时有声书库被过滤掉，导致有声书页面显示 0 本。

**原因：** `scanner/scanner.go` 第 98-101 行在全量扫描时过滤掉了 `MediaType == "audiobook"` 的库，但 `scanAudiobooks()` 又从已过滤的列表中查找有声书库，导致找不到。

**修复：** `scanAudiobooks()` 改为直接从数据库获取所有库，不依赖已过滤的 `state.libraries`。

**Commit：** `99bb7a4c`（LeChenMusic-Server）
