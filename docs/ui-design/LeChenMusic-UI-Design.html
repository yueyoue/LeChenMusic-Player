12 组件规范
可复用组件的设计规格，确保还原度 100%

卡片组件

// 影视卡片规格
手机宽度: 130px (横向滚动) / 自适应 33% (网格)
宽高比: 2:3 (海报) / 16:9 (缩略图)
圆角: 12px
间距: 10px (横向) / 12px (网格)
评分标签: 右上角, 红底白字, 11px, 6px圆角
集数标签: 左下角, 黑底白字, 10px, 4px圆角

// 音乐歌曲行规格
高度: 48px 封面 + 文字
封面: 48x48px, 8px圆角
歌名: 14px Semibold
歌手: 12px, #8888AA
时长: 12px, #555577

按钮组件

// 主按钮
背景: accent-gradient (135deg, #6C5CE7, #A29BFE)
文字: 白色, 15px Semibold
圆角: 12px
内边距: 14px 24px

// 次按钮
背景: 透明
边框: 1px solid rgba(255,255,255,0.06)
文字: #F0F0F5, 13px Medium
圆角: 12px

// Filter Pill
背景: rgba(255,255,255,0.04)
边框: 1px solid rgba(255,255,255,0.06)
圆角: 20px (胶囊)
选中态: accent 背景, 白色文字

导航栏组件

// 底部导航栏 (手机)
高度: 64px
背景: rgba(10,10,15,0.95) + backdrop-filter blur(20px)
分割线: 1px solid rgba(255,255,255,0.06)
图标: 22px, 未选中 50% 透明度
文字: 10px, 选中时 accent-light + 600 weight

// 侧边导航 Rail (平板)
宽度: 80px
背景: 同底部导航
项目: 64px宽, 10px 8px padding, 12px 圆角
选中态: rgba(108,92,231,0.15) 背景
13 给开发者的实现指南
技术栈、组件映射、关键实现细节
🛠️ 技术栈
UI 框架: Jetpack Compose + Material3
响应式: material3-window-size-class
图片加载: Coil
导航: Navigation Compose
播放器: ExoPlayer (Media3)
投屏: 自研 SSDP + UPnP (OkHttp)

Compose 组件映射

// 设计稿 → Compose 组件对照

底部导航 → NavigationBar + NavigationBarItem
侧边导航 → NavigationRail + NavigationRailItem
搜索栏 → TextField(OutlinedTextFieldDefaults)
Tab 栏 → ScrollableTabRow + Tab
横向滚动 → LazyRow
网格 → LazyVerticalGrid(GridCells.Fixed(n))
卡片 → Card / Surface(shape=RoundedCornerShape)
筛选 Pill → FilterChip / Surface + clickable
底部弹窗 → ModalBottomSheet
迷你播放器 → Surface + Row(固定在底部)
毛玻璃 → Modifier.background(color.copy(alpha))

// 响应式网格列数
val gridColumns = when (windowSizeClass.widthSizeClass) {
  Compact → 3
  Medium → 5
  Expanded → 6
}
