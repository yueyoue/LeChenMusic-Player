## 项目概述

**LeChenMusic** 是一个音乐+有声书+影视的 Android APP(Jetpack Compose),服务端基于 Navidrome/Subsonic 协议,影视功能对接 LunaTV。

- **APP端仓库**: https://github.com/yueyoue/LeChenMusic-Player
- **服务端仓库**: https://github.com/yueyoue/LeChenMusic-Server
- **LunaTV 服务**: http://j.tthsdd.top:3000 (admin / shaozi1981)
- **LunaTV 源码**: https://github.com/MoonTechLab/LunaTV
- **参考APP(Selene-Source)**: https://github.com/MoonTechLab/Selene-Source (Flutter,为 LunaTV 配套)

**核心原则**: 所有影视功能直接参考 Selene-Source 的实现方法。遇到问题第一时间看 Selene-Source 的解决方案。

## 一、LunaTV 实际 API(已验证)

⚠️ **LunaTV 返回的是裸 JSON,没有 code/msg/data 包装!**

| 接口 | 方法 | 实际返回格式 | 注意事项 |
|------|------|-------------|---------|
| `/api/login` | POST | `{"ok": true}` | body: `{"username":"admin","password":"***"}` |
| `/api/search` | GET | `{"results": [VideoInfo...]}` | **参数用 `q` 不是 `keyword`** |
| `/api/detail` | GET | `VideoDetail` 裸对象 | 参数: `source`, `id` |
| `/api/playrecords` | GET | `{"source+id": {record...}}` | **是 dict 不是 list** |
| `/api/playrecords` | POST | 保存播放记录 | **body: `{"key":"source+id", "record":{...}}`** |
| `/api/favorites` | GET | `[VideoInfo...]` | 裸数组 |
| `/api/search/resources` | GET | `[SearchResource...]` | 裸数组 |
| `/api/live/sources` | GET | `[LiveSource...]` | - |
| `/api/live/channels` | GET | `[LiveChannelGroup...]` | - |

**不存在的接口**(返回 404): `/api/home`, `/api/category`

### 搜索结果 VideoInfo 关键字段
```json
{
  "id": "65490", "title": "痴迷2025", "poster": "https://...",
  "episodes": ["https://xxx.m3u8", ...],
  "episodes_titles": ["1", "2", ...],
  "source": "iqiyizyapi.com", "source_name": "🎬-爱奇艺-",
  "year": "2025", "type_name": "movie", "douban_id": 0
}
```

### 豆瓣 API(已验证)

| 接口 | 说明 | 注意事项 |
|------|------|---------|
| `v2/subject/recent_hot/{kind}` | 热门推荐 | `kind`=movie/tv/show/anime |
| `v2/{kind}/recommend` | 推荐(带筛选) | `tags` 逗号分隔,但 `tags=热门` 返回空! |

**⚠️ 豆瓣图片**: `img9.doubanio.com` 被拦截(418),必须替换为 `img3.doubanio.com` + Referer 头

## 六、给下位程序员的关键提示 ⚠️

### 绝对不要踩的坑

| 坑 | 说明 | 后果 |
|----|------|------|
| **ExoPlayer HLS** | 必须有 `media3-exoplayer-hls` | native 崩溃 |
| **LunaTV 搜索参数** | 用 `q` 不是 `keyword` | 搜索返回空 |
| **播放记录保存格式** | `{"key":"source+id","record":{...}}` | 保存无效 |
| **豆瓣图片** | `img9.doubanio.com` 被拦截 | 图片全白 |
| **豆瓣 recommend+tags=热门** | 返回空结果! | 热门电影消失 |
| **LaunchedEffect 位置** | 不要放 LazyColumn item 内部 | 滚动时被回收 |
| **ExoPlayer seekTo** | prepare 后不能立即 seek | 必须等 STATE_READY |
| **源切换** | 不要依赖 LaunchedEffect | 直接控制 ExoPlayer |

### 修改代码的核心原则

**直接控制 ExoPlayer**:
```kotlin
// ✅ 正确: 直接在 onClick 中控制
exoPlayer.stop()
exoPlayer.setMediaItem(MediaItem.fromUri(url))
exoPlayer.prepare()
exoPlayer.playWhenReady = true

// ❌ 错误: 通过 ViewModel 状态间接触发
viewModel.switchSource(info)  // 有延迟和竞争条件!
```

**搜索用精确匹配**:
```kotlin
// ✅ 正确
val normalizedTitle = title.replace(" ", "").lowercase()
val matched = sources.firstOrNull { it.title.replace(" ", "").lowercase() == normalizedTitle }

// ❌ 错误
val matched = sources.firstOrNull { it.title.contains(title, ignoreCase = true) }
```
